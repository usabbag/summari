Summariz.io

Dense, meaningful AI summaries of any page in seconds

+Summarize ↑ Drag to your bookmarks bar

Quick Setup

  1. Get your OpenAI API key from platform.openai.com
  2. Drag the Summarize button to your bookmarks bar
  3. Click the bookmark on any webpage - you'll be asked for your API key the first time

Note: Your API key is securely stored in your browser - you'll only need to enter it once.

Features

Fast & Easy

Run this bookmarklet on any browser to get an instant summary. No app to install.

Dual Extraction Methods

The bookmarklet can summarize content using either the page URL or the text on the page, ensuring flexibility and accuracy.

Opinionated Summaries

We've crafted the perfect prompt to give you dense, 200-word summaries that actually explain what matters. No bullet points, just clear insights.

Installation Guide

Mac

  1. Show bookmarks bar with ⌘+Shift+B
  2. Drag the Summarize button to your bookmarks bar
  3. Enter your API key when prompted (first use only)

iOS

  1. Copy the Summarize button link
  2. Create a new bookmark in Safari
  3. Paste the copied link as the bookmark URL
  4. Enter your API key when prompted (first use only)
